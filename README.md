# Tournament System

This is a simple gem that implements numerous tournament systems.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tournament-system'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tournament-system
